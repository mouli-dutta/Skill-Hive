## 💾 Your Data

### Automatic Saving
Everything is saved automatically to your browser:
- Progress and completions
- XP and level
- Streak data
- Achievements
- Settings and preferences

### Backup Your Progress
1. Go to **Settings**
2. Click **"Export Progress"**
3. Save the JSON file
4. To restore: **"Import Progress"**

### Reset Progress
Settings → **"Reset All Progress"** (use with caution!)

## 🆘 Troubleshooting

### App Not Loading?
- Make sure you're opening `index.html` in a modern browser
- Try Chrome, Firefox, Safari, or Edge
- Check browser console for errors (F12)

### Progress Not Saving?
- Ensure localStorage is enabled
- Check browser privacy settings
- Try exporting/importing your data

### Theme Not Switching?
- Click the sun/moon icon (top right)
- Check Settings → Theme
- Try refreshing the page

### Something Broken?
- Refresh the page (F5)
- Clear browser cache
- Try a different browser
- Reset progress (last resort)
